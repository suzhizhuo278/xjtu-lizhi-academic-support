# 励志学辅资料集

用于西安交大励志书院学业辅导中心资料的统一整理与存储。

## 项目结构（仅 3 个一级目录）

```text
.
├── 中心说明/
├── 学习资源/
└── 模板/
```

## 各目录用途

- `中心说明/`：中心介绍、值班安排、制度、联系人等固定说明。
- `学习资源/`：所有可供同学使用的学习资料。
- `模板/`：通知、复盘、资料说明等模板文件。

## 学习资源的统一格式（每个资料包）

每个资料包统一为：

```text
xx资料/
├── pdf/
├── src/
├── graph/
└── README.md
```

说明：
- `pdf/`：最终分发版本。
- `src/`：可编辑源文件（Word/PPT/Markdown 等）。
- `graph/`：图片、图表、截图。
- `README.md`：该资料包简介、内容清单、维护人、更新时间。

## 命名建议

统一格式：`YYYY-MM-DD_主题_版本`  
示例：`2026-03-16_高数A1期中答疑_v1.pdf`
