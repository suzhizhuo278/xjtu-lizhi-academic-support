# 励志学辅资料集

用于西安交大励志书院学业辅导中心资料的统一整理与存储。

## 建议项目结构（极简版）

```text
.
├── 课程资料/
│   ├── 高数A1资料/
│   │   ├── pdf/      # 成品资料（讲义、题单、答案、手册）
│   │   ├── src/      # 可编辑源文件（Word、PPT、Markdown、LaTeX等）
│   │   ├── graph/    # 图片与图表（流程图、截图、配图）
│   │   └── README.md # 本资料说明（内容、来源、维护人、更新记录）
│   └── 大学物理资料/
│       ├── pdf/
│       ├── src/
│       ├── graph/
│       └── README.md
├── 通用资源/
│   ├── 学习方法资料/
│   │   ├── pdf/
│   │   ├── src/
│   │   ├── graph/
│   │   └── README.md
│   └── 工具指南资料/
```

## 统一规则（重点）

### 1) 每个“资料包”固定四件套
每个资料都按以下结构创建：

- `xx资料/pdf`
- `xx资料/src`
- `xx资料/graph`
- `xx资料/README.md`

### 2) 文件命名建议
统一格式：`YYYY-MM-DD_主题_版本`  
示例：`2026-03-16_高数A1期中答疑_v1.pdf`

### 3) README.md（资料内）建议写什么
每个 `xx资料/README.md` 建议包含：

- 资料简介（这份资料是干什么的）
- 内容清单（pdf/src/graph 中各文件作用）
- 来源与版权说明（是否可对外分享）
- 维护人（姓名+联系方式）
- 更新记录（日期 + 变更说明）

### 4) 最小维护原则

- 只保留一个最新版，历史版本放 `archive/`。
- 新增资料时，优先补齐 `README.md`。
- 先放 `src` 再导出 `pdf`，避免后续无法修改。
